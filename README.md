# Каталог сказок и историй

Интерактивный литературный каталог сказок и историй из журнала
[gotli.livejournal.com](https://gotli.livejournal.com) (тег «сказки» / «волшебное»):
80 рассказов по 7 циклам, галерея сквозных персонажей, две сцены,
полнотекстовый поиск с подсветкой, избранное, кросс-ссылки между рассказами
и лайтбокс для изображений.

Весь сайт — один самодостаточный файл `index.html` (все изображения встроены
внутрь как WebP/base64, внешних зависимостей нет).

## Как открыть

Просто откройте `index.html` в браузере — или опубликуйте через GitHub Pages
(см. инструкцию, которая шла вместе с файлом).
